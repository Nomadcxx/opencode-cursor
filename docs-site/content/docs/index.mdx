open-cursor connects OpenCode to the models available through your Cursor
subscription. It translates prompts, streaming responses, and tool calls
between OpenCode and `cursor-agent`.

## Getting started

- [Installation](/docs/getting-started/installation) sets up the package,
  OpenCode provider, and current Cursor model list.
- [Authentication](/docs/getting-started/authentication) covers Cursor CLI
  login, API keys, and checks for each backend.
- [Quick start](/docs/getting-started/quick-start) runs a first prompt and
  explains automatic model selection.
- [Troubleshooting](/docs/getting-started/troubleshooting) maps common symptoms
